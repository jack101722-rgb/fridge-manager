+ 버튼을 누르면 이 시트가 올라와요. 5가지 방법 중 편한 걸 골라 등록하세요.

1

✏️ 직접 입력 — 재료 이름, 카테고리, 수량, 소비기한을 직접 써요. 기한을 모르면 비워도 AI가 자동 예측해줘요.

2

📷 카메라 촬영 — 재료 사진을 찍으면 AI가 자동으로 재료 이름을 인식해요.

3

🧾 영수증 스캔 — 마트 영수증을 찍으면 구매 항목을 한 번에 등록해줘요. 여러 개를 한꺼번에 넣을 때 최고예요.

4

📦 바코드 스캔 — 제품 뒷면 바코드를 스캔하면 상품 정보를 자동으로 불러와요.

5

🗂️ 패키지 선택 — 찌개 재료, 샐러드 재료처럼 자주 쓰는 재료 묶음을 한 번에 등록해요.

💡 재료를 탭하면 수량 수정, 부분 소비, 삭제가 가능해요.

하단 탭에서 🍳 메뉴를 누르면 열려요. 재료 5개 이상 등록하면 활성화돼요.

A

카테고리별 추천 — 계절 추천 / 10분 간단 요리 / 한식 / 양식 중 원하는 걸 골라볼 수 있어요.

B

내 냉장고 맞춤 추천 — 지금 내 냉장고에 있는 재료를 기반으로 AI가 실시간으로 메뉴를 골라줘요.

C

메뉴를 탭하면 상세 레시피가 펼쳐져요 — 재료 목록, 조리 순서, 조리 팁까지 AI가 맞춤으로 생성해줘요.

💡 메뉴 카드에 "✅ 보유" 표시가 있으면 지금 바로 만들 수 있는 재료가 있다는 뜻이에요.

하단 탭에서 🛒 장보기를 눌러요. 마트에 가기 전 목록을 만들고, 사고 나면 바로 냉장고에 넣을 수 있어요.

1

항목 추가 — 살 재료를 직접 입력하거나, AI가 메뉴 레시피 기반으로 자동으로 추가해줘요.

2

구매 체크 — 마트에서 담을 때마다 동그라미를 탭해서 체크하세요. 완료된 항목은 아래로 내려가요.

3

하단 "냉장고에 추가하기" 버튼을 누르면 — 구매 완료 항목이 한 번에 냉장고로 들어가요. AI가 소비기한 등 정보를 자동으로 채워줘요.

💡 장보기 항목 옆에 출처(레시피명)가 표시돼요. 어떤 요리 때문에 사야 하는지 마트에서 바로 확인할 수 있어요.
